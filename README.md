# get-stuff-done
A basic Flask CRUD app for managing your ToDo list.

## Installation
To use this app, install:

```bash
python
pip3 install virtualenv 
virtualenv env 
```

## Activate env
```bash
 source env/bin/activate  
```
 
## Run
```bash
pip3 install flask flask-sqlalchemy
```

## Serve App
```bash
python3 app.py
```
